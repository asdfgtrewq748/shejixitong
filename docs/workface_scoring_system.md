## 概述

采矿工程设计是一个复杂的系统工程，需要综合考虑安全、经济、技术、环保等多个维度。本评分体系针对工作面设计方案，从**8个主要维度**进行综合评价，确保设计方案的科学性和合理性。

---

## 评分维度

### 评分框架

```
工作面综合评分 (100分)
├── 1. 地质条件评分 (20分) - 基于地质网格评分
├── 2. 安全性评分 (20分) - 考虑采深、煤柱、瓦斯等
├── 3. 经济性评分 (15分) - 考虑开采成本、效率、回采率
├── 4. 技术可行性评分 (15分) - 考虑设备适应性、施工条件
├── 5. 开采效率评分 (10分) - 考虑工作面几何参数
├── 6. 通风条件评分 (10分) - 考虑巷道布局、瓦斯管理
├── 7. 运输条件评分 (5分) - 考虑运输距离、巷道坡度
└── 8. 环保影响评分 (5分) - 考虑地表沉陷、水害风险
```

---

## 评分指标详解

### 1. 地质条件评分 (20分)

基于地质评分网格，综合考虑工作面范围内的地质条件。

#### 1.1 指标构成
- **煤层厚度均匀性** (8分)
  - 厚度变异系数 < 0.1：满分
  - 厚度变异系数 0.1-0.2：6分
  - 厚度变异系数 0.2-0.3：4分
  - 厚度变异系数 > 0.3：2分

- **煤层倾角稳定性** (6分)
  - 倾角变化 < 3°：满分
  - 倾角变化 3-5°：4分
  - 倾角变化 5-8°：2分
  - 倾角变化 > 8°：0分

- **地质构造复杂度** (6分)
  - 无断层或褶皱：满分
  - 小型断层（落差<3m）：4分
  - 中型断层（落差3-10m）：2分
  - 大型断层（落差>10m）：0分

#### 1.2 计算公式
```javascript
地质条件评分 = 工作面范围内地质网格平均分 * 0.2
```

---

### 2. 安全性评分 (20分)

#### 2.1 指标构成

**2.1.1 采深安全系数** (8分)
```javascript
if (采深 < 300m) 分数 = 8;
else if (采深 < 500m) 分数 = 7;
else if (采深 < 700m) 分数 = 6;
else if (采深 < 900m) 分数 = 4;
else 分数 = 2;
```

**2.1.2 煤柱宽度合规性** (6分)
```javascript
煤柱合规度 = 实际煤柱宽度 / 设计煤柱宽度;
if (合规度 >= 1.0 && 合规度 <= 1.2) 分数 = 6;  // 刚好或略宽
else if (合规度 >= 0.9 && 合规度 < 1.0) 分数 = 4;  // 略窄
else if (合规度 > 1.2 && 合规度 <= 1.5) 分数 = 4;  // 过宽浪费
else 分数 = 0;  // 严重不合规
```

**2.1.3 瓦斯风险** (3分)
```javascript
平均瓦斯含量 = 工作面范围内钻孔瓦斯含量平均值;
if (瓦斯 < 5m³/t) 分数 = 3;  // 低瓦斯
else if (瓦斯 < 10m³/t) 分数 = 2;  // 中瓦斯
else 分数 = 1;  // 高瓦斯
```

**2.1.4 顶板稳定性** (3分)
```javascript
平均岩石硬度 = 工作面范围内顶板岩石硬度平均值;
if (硬度 >= 6 && 硬度 <= 8) 分数 = 3;  // 理想范围
else if (硬度 >= 4 && 硬度 < 6) 分数 = 2;  // 偏软
else if (硬度 > 8) 分数 = 2;  // 过硬
else 分数 = 0;  // 极软，不稳定
```

---

### 3. 经济性评分 (15分)

#### 3.1 指标构成

**3.1.1 开采成本指数** (6分)
```javascript
成本指数 = (采深/100) * 1.5 + (平均倾角/45) * 2.0 + (煤层厚度变异系数) * 3.0;
// 成本指数越低越好
if (成本指数 < 5) 分数 = 6;
else if (成本指数 < 8) 分数 = 4;
else if (成本指数 < 12) 分数 = 2;
else 分数 = 0;
```

**3.1.2 回采率** (5分)
```javascript
回采率 = 工作面面积 / (工作面面积 + 相关煤柱面积);
分数 = 回采率 * 5;  // 例如：80%回采率 = 4分
```

**3.1.3 煤质经济价值** (4分)
```javascript
平均发热量 = 工作面范围内煤质发热量平均值;
if (发热量 > 25 MJ/kg) 分数 = 4;  // 优质煤
else if (发热量 > 20 MJ/kg) 分数 = 3;  // 良煤
else if (发热量 > 16 MJ/kg) 分数 = 2;  // 中煤
else 分数 = 1;  // 劣煤
```

---

### 4. 技术可行性评分 (15分)

#### 4.1 指标构成

**4.1.1 工作面宽度适应性** (5分)
```javascript
// 综采设备标准工作面宽度：150-200m
if (宽度 >= 150 && 宽度 <= 200) 分数 = 5;
else if (宽度 >= 120 && 宽度 < 150) 分数 = 4;
else if (宽度 >= 100 && 宽度 < 120) 分数 = 3;
else if (宽度 > 200 && 宽度 <= 250) 分数 = 3;
else 分数 = 1;
```

**4.1.2 煤层厚度适应性** (5分)
```javascript
平均厚度 = 工作面范围内煤层厚度平均值;
if (厚度 >= 1.3 && 厚度 <= 3.5) 分数 = 5;  // 中厚煤层，最佳
else if (厚度 > 3.5 && 厚度 <= 6) 分数 = 4;  // 厚煤层
else if (厚度 > 0.8 && 厚度 < 1.3) 分数 = 3;  // 薄煤层
else 分数 = 1;
```

**4.1.3 倾角适应性** (5分)
```javascript
平均倾角 = 工作面范围内倾角平均值;
if (倾角 <= 8°) 分数 = 5;  // 近水平
else if (倾角 <= 15°) 分数 = 4;  // 缓倾斜
else if (倾角 <= 25°) 分数 = 3;  // 倾斜
else if (倾角 <= 45°) 分数 = 1;  // 急倾斜
else 分数 = 0;  // 极陡
```

---

### 5. 开采效率评分 (10分)

#### 5.1 指标构成

**5.1.1 工作面面积** (4分)
```javascript
面积 = 宽度 × 推进长度;
// 面积越大，单产越高
if (面积 > 100000) 分数 = 4;
else if (面积 > 80000) 分数 = 3;
else if (面积 > 60000) 分数 = 2;
else 分数 = 1;
```

**5.1.2 长宽比合理性** (3分)
```javascript
长宽比 = 推进长度 / 工作面宽度;
// 理想范围：4-8
if (长宽比 >= 4 && 长宽比 <= 8) 分数 = 3;
else if (长宽比 >= 3 && 长宽比 < 4) 分数 = 2;
else if (长宽比 > 8 && 长宽比 <= 10) 分数 = 2;
else 分数 = 1;
```

**5.1.3 可采储量** (3分)
```javascript
可采储量 = 面积 × 平均厚度 × 煤层密度(1.4t/m³);
// 储量越大，开采周期越长
if (储量 > 200000吨) 分数 = 3;
else if (储量 > 150000吨) 分数 = 2;
else 分数 = 1;
```

---

### 6. 通风条件评分 (10分)

#### 6.1 指标构成

**6.1.1 巷道通风距离** (4分)
```javascript
通风路径长度 = 工作面到回风巷的距离;
if (长度 < 1000m) 分数 = 4;
else if (长度 < 1500m) 分数 = 3;
else if (长度 < 2000m) 分数 = 2;
else 分数 = 1;
```

**6.1.2 瓦斯排放能力** (3分)
```javascript
// 基于工作面宽度和瓦斯含量
瓦斯排放需求 = 平均瓦斯含量 × 工作面宽度;
if (需求 < 500) 分数 = 3;
else if (需求 < 1000) 分数 = 2;
else 分数 = 1;
```

**6.1.3 通风系统布局** (3分)
```javascript
// U型通风系统（最佳）
if (有独立进风和回风巷) 分数 = 3;
else if (共用部分巷道) 分数 = 2;
else 分数 = 1;
```

---

### 7. 运输条件评分 (5分)

#### 7.1 指标构成

**7.1.1 运输距离** (3分)
```javascript
平均运输距离 = 工作面中心到主运输大巷的距离;
if (距离 < 500m) 分数 = 3;
else if (距离 < 1000m) 分数 = 2;
else 分数 = 1;
```

**7.1.2 巷道坡度** (2分)
```javascript
// 理想坡度：小于5°
if (平均坡度 < 5°) 分数 = 2;
else if (平均坡度 < 10°) 分数 = 1;
else 分数 = 0;
```

---

### 8. 环保影响评分 (5分)

#### 8.1 指标构成

**8.1.1 地表沉陷风险** (3分)
```javascript
采厚采深比 = 煤层厚度 / 采深;
if (采厚采深比 < 1/100) 分数 = 3;  // 低沉陷风险
else if (采厚采深比 < 1/50) 分数 = 2;
else 分数 = 1;
```

**8.1.2 水害风险** (2分)
```javascript
// 基于钻孔富水指数
平均富水指数 = 工作面范围内钻孔富水指数平均值;
if (富水指数 < 5) 分数 = 2;
else if (富水指数 < 10) 分数 = 1;
else 分数 = 0;
```

## 计算方法

### 综合评分公式

```javascript
工作面综合评分 = Σ(各维度得分)

总分 = 地质条件(20) + 安全性(20) + 经济性(15) + 技术可行性(15)
     + 开采效率(10) + 通风条件(10) + 运输条件(5) + 环保影响(5)
```

### 权重调整（可选）

根据矿井实际情况，可以调整各维度权重：

```javascript
// 高瓦斯矿井：增加安全性和通风权重
安全性权重 = 25%, 通风条件权重 = 15%, 其他维度相应减少

// 经济困难矿井：增加经济性权重
经济性权重 = 25%, 其他维度相应减少

// 深部开采：增加安全性和技术可行性权重
安全性权重 = 25%, 技术可行性权重 = 20%
```

## 评分标准

### 等级划分

| 得分范围 | 等级 | 评价 | 建议 |
|---------|------|------|------|
| 90-100 | A+ | 优秀 | 优先开采，推荐设计方案 |
| 80-89 | A | 良好 | 可以开采，局部可优化 |
| 70-79 | B+ | 较好 | 可以开采，建议优化部分设计 |
| 60-69 | B | 一般 | 需要改进设计方案 |
| 50-59 | C | 较差 | 不推荐，需重大改进 |
| < 50 | D | 很差 | 不建议开采 |

### 单项否决指标

以下情况下，无论综合评分多高，都应**否决**该工作面设计：

1. **安全性评分 < 10分** - 存在重大安全隐患
2. **煤柱宽度不合规（<90%设计值）** - 严重违反规程
3. **大型断层（落差>10m）穿过工作面** - 无法开采
4. **高突风险区域（瓦斯>15m³/t且深度>500m）** - 需要特殊措施

## 应用示例

### 示例1：浅部中厚煤层工作面

**基本参数：**
- 采深：350m
- 煤层厚度：2.5m（变异系数0.08）
- 倾角：8°（变化<2°）
- 工作面宽度：180m
- 推进长度：1000m
- 瓦斯含量：6 m³/t
- 顶板硬度：7
- 发热量：24 MJ/kg

**评分计算：**

1. **地质条件** = 18/20
   - 厚度均匀性：8分（变异系数<0.1）
   - 倾角稳定性：6分（变化<3°）
   - 无构造：6分（无断层）

2. **安全性** = 17/20
   - 采深：7分（<500m）
   - 煤柱合规：6分（合规）
   - 瓦斯：2分（中瓦斯）
   - 顶板：3分（理想硬度）

3. **经济性** = 13/15
   - 成本指数：5分（成本较低）
   - 回采率：4分（80%）
   - 煤质：4分（优质煤）

4. **技术可行性** = 15/15
   - 宽度适应性：5分（标准范围）
   - 厚度适应性：5分（中厚煤层）
   - 倾角适应性：5分（近水平）

5. **开采效率** = 9/10
   - 面积：4分（180000m²）
   - 长宽比：3分（5.56，理想）
   - 储量：3分（>200000吨）

6. **通风条件** = 8/10
   - 通风距离：3分（<1000m）
   - 瓦斯排放：2分（中等）
   - 通风布局：3分（U型）

7. **运输条件** = 4/5
   - 运输距离：2分（<1000m）
   - 巷道坡度：2分（<5°）

8. **环保影响** = 4/5
   - 沉陷风险：3分（采厚采深比1/140）
   - 水害风险：2分（低富水）

**综合评分：88分（A级，良好）**

## 实施建议

### 1. 数据采集要求

为准确评分，需要采集以下数据：
- 钻孔地质数据（煤层厚度、倾角、岩性、瓦斯、水文）
- 采区边界和地形
- 主巷道位置
- 设备参数（综采设备能力）
- 环保敏感区位置

### 2. 评分流程

```
设计方案输入
    ↓
提取工作面参数
    ↓
计算各维度指标
    ↓
加权汇总得分
    ↓
应用否决条件
    ↓
输出评分和等级
```

### 3. 评分应用

- **设计阶段**：用于方案比选和优化
- **审查阶段**：用于设计审批依据
- **调整阶段**：识别薄弱环节，针对性改进
- **监控阶段**：实际开采中对比预期评分
